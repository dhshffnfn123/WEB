# Margin

  • 해당 요소의 border 바깥쪽의 여백
  • 해당 요소와 다른 요소와의 거리를 결정한다
  • top margin과 bottom margin이 만나는 곳에서는 둘 중 더 높은 margin만 적용된다
  • margin을 auto로 설정하면 자동으로 margin을 조절하여 요소를 가운데로 정렬한다

# Box Model

# Box Sizing

  • content-box는 width + padding + border으로 크기를 결정한다
  • border-box는 width만으로 요소의 크기를 결정한다
