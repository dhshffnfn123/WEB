# 가상 클래스 (Pseudo Class)

  • 선택자에 가상 클래스를 사용하여 해당요소의 특정 상황만을 선택할 수 있따
  • 선택자 뒤에 : 을 붙여 원하는 상황을 선택한다

# 가상 클래스들

firtst-child, nth-child(n/evne/odd), last-child
- 요소가 부모의 특정 번째 요소일 때를 선택함
first-of-typem nth-of-type(n/even/odd), last-of-type
- 요소가 같은 타입들 중에서 특정 번째 요소일 때를 선택함
hover
- 요소에 마우스가 올라가 있는 상황을 선택함
link
- 해당 요소가 링크일 때를 선택한다
active
- 해당 링크가 마우스로 활성화되고 있는 상황을 선택한다
visited
- 해당 링크가 이미 방문했던 링크일 때를 선택한다
