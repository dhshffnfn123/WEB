# 가상 요소 (Pseudo Element)

  • 가상 요소는 해당 요소의 특정 부분을 선택할 수 있다
  • 선택자 뒤에 ::을 붙여 원하는 부분만 선택한다
  • ::first_letter
  • ::first_line
  • ::before
  • ::after
