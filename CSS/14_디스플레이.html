# display property

  • 해당 요소가 어떻게 보여야 할지를 결정하는 속성
  • 이 속성의 기본값으로는 inline과 block이 있다.

# inline 요소의 특징

  • 문단 내부에서 영역을 지정할 때 사용한다
  • 새 줄에서 시작하지 않고, 필요한 내용만큼의 영역을 차지한다
  • padding과 margin을 고려하지 않고 배치된다

# block 요소의 특징

  • 항상 새 줄에서 시작한다
  • 기본 너비는 100%여서 화면 끝까지 뻗어 나간다
  • padding과 margin을 고려하지 않고 배치된다
  • 박스 취급

# display: inline-block

  • 글자 취급 받지만 padding과 margin이 적용된다
  • inline과 block의 단점을 모두 보완한 display값이다
# inline test

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Etiam dignissim arcu nulla, in ultrices diam imperdiet eu.

# block test

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Etiam dignissim arcu nulla, in ultrices diam imperdiet eu.

# inline-block test

Lorem ipsum dolor sit amet, consectetur adipiscing elit. Etiam dignissim arcu nulla, in ultrices diam imperdiet eu.

# display: none

해당 요소를 보이지 않게 숨긴다 (영역도 함께 사라진다)

보이지 않는 요소

# visibility: hidden

해당 요소를 보이지 않게 숨긴다 (영역은 그대로 유지된다)

보이지 않는 요소
