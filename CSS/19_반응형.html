# 반응형 웹 사이트

  • viewport의 크기에 따라 스타일이 달라지는 웹 사이트
  • Media Query를 사용한다
  • @media (미디어 타입) and (원하는 크기)

# Media Types

  • all : 모든 타입
  • print : 프린터기에서 볼 때의 타입
  • screen : 컴퓨터 스크린, 태블릿, 스마트폰 등의 일반 기기
  • speech : 화면을 읽어주는 기기
